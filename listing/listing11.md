Что выведет данная программа и почему?

```go
func main() {
    wg := sync.WaitGroup{}
    for i := 0; i < 5; i++ {
        wg.Add(1)
        go func(wg sync.WaitGroup, i int) {
            fmt.Println(i)
            wg.Done()
        }(wg, i)
	}
    wg.Wait()
    fmt.Println("exit")
}

```

Ответ:

Будет выведено 5 значений от 0 до 4 в произвольном порядке, после чего произойдет deadlock. Сообщение "exit" не будет выведено. 
Происходит это из-за того, что в горутину передается копия объекта sync.WaitGroup, а не ссылка на него. 
Поэтому вызов wg.Done() не уменьшает счетчик ожидания, и программа виснет в ожидании завершения всех горутин.
